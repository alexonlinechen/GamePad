# GamePad
ESP32 遊戲搖桿

模式1(鍵盤方向鍵) 
模式2(鍵盤WASD)  
模式3(滑鼠模式)

切換模式  按鍵1+2+3+4

模式1(鍵盤方向鍵) 

搖桿 == 鍵盤方向鍵

按鍵1 --->  ESC鍵

按鍵2 --->  ENTER鍵

按鍵3 --->  BACKSPACE鍵

按鍵4 --->  END鍵

按鍵5 --->  PAGE_UP鍵

按鍵6 --->  PAGE_DOWN鍵

按鍵7 --->  ESC鍵

左側按鍵 --->  LEFT_SHIFT鍵

右側按鍵 --->  RIGHT_SHIFT鍵




模式2(鍵盤WASD鍵) 

搖桿 == WASD鍵</br>
按鍵1 --->  ESC鍵

按鍵2 --->  ENTER鍵

按鍵3 --->  BACKSPACE鍵

按鍵4 --->  END鍵

按鍵5 --->  PAGE_UP鍵

按鍵6 --->  PAGE_DOWN鍵

按鍵7 --->  ESC鍵

左側按鍵 --->  LEFT_SHIFT鍵

右側按鍵 --->  RIGHT_SHIFT鍵


模式3(滑鼠模式)

搖桿 == 滑鼠游標

按鍵3 --->  滑鼠左鍵  / 確認鍵

按鍵4 --->  滑鼠右鍵 / 返回鍵
